# v5 已修复/新增内容一览：

1. 漏洞扫描与修复：
    - 上传仅允许真实图片（getimagesize + MIME + 扩展名校验），杜绝 .php/.phtml 等脚本远程执行。
    - 文件名强制净化（只保留字母数字 _ - .），+ 时间戳前缀，彻底杜绝同名覆盖或路径穿越。
    - 所有路径强制在 pics/ 内，绝不会触碰 web 目录以外任何文件。
2. 防止损坏外部文件：已用 realpath + 严格路径检查 + basename 防护。
3. 编辑页多选删除：每张图加复选框 + “全选” + “批量删除选中”按钮。
4. 详情页排序：顶部新增下拉菜单，支持文件名正/逆序、文件写入时间正/逆序、随机打乱（纯临时，刷新恢复默认，不保存）。
5. 编辑页上传按钮居中做大：与首页完全一致的大按钮，手机超好点。
6. 编辑页手动排序：每张图增加 ↑ ↓ 按钮（点击即可上下移动），实时预览顺序，点“保存当前顺序”后永久保存（使用 order.json，最轻量、无扩展、无累赘，复制整个网站到任何 PHP 服务器无缝运行）。
7. 详情页  编辑页自由跳转：详情页新增“编辑此照片墙”按钮，编辑页也有“查看照片墙”按钮。

8. 极端情况防护：
    - 同名不同内容图片：自动加时间戳永不覆盖。
    - 恶意文件名：全部净化。
    - 非图片上传：自动跳过并提示。
    - 空文件夹、权限问题等均有安全处理。
9. 存储方式：使用 order.json（JSON 文件），无需 SQLite、无 PDO、无额外扩展，复制粘贴即可在任意 PHP 服务器运行。
